## 二、关键节点数据率与格式

| 节点 | 数据率 | 每帧数据量 | 说明 |
|------|--------|-----------|------|
| 信息比特 (Info Bits) | 2 kbps | 1998 bit | 原始信息，randi([0,1]) |
| 卷积编码后 (Encoded) | 4 kbps | 4000 bit | (7,5) 1/2码率，自动尾比特 |
| 交织后 (Interleaved) | 4 kbps | 4000 bit | 50×80 块交织，零填充 |
| GFSK调制符号 | 1 kSps | 2000 sym | 4FSK，h=0.5，BT=0.5 |
| 上采样后 | 64 kSps | 32000 sample | sps=16 |
| 射频采样 | 32 MSps | 64M sample | 复指数载波 |
| CIC抽取后 | 64 kSps | — | 2×2×2×125 = 1000× |
| 半带抽取后 | 16 kSps | — | 4× |
| 基带复信号 | 16 kSps | 16000 sample | sps_rx = 8 |
| 4路混频+LPF | 16 kSps × 4 | 64000 sample | ±500Hz, ±1500Hz |
| 符号采样点 | 1 kSps | 1000 sample | sps_rx/2 = 4 |
| 硬判决比特 | 2 kbps | 2000 bit | 4电平Max判决定 |
| 解交织后 | 4 kbps | 4000 bit | 恢复原始编码序列 |
| 维特比译码后 | 2 kbps | 1998 bit | 信息比特级BER统计 |

## 四、(7,5) 卷积码细节

### 4.1 编码器结构

```
               ┌─────────────────────────────────────┐
               │         (7,5) 卷积编码器            │
               │        K=3, 码率 R=1/2              │
               └─────────────────────────────────────┘

    输入比特 d(k) ─────────────────┬──────────────────────────→ 输出 u1(k)
                                   │
                                   ▼
                             ┌─────────┐
                             │  D      │ ← 寄存器1 (S1)
                             └────┬────┘
                                  │
                                  ▼
                             ┌─────────┐
                             │  D      │ ← 寄存器2 (S2)
                             └────┬────┘
                                  │
                                  └──────┬───────────────────→ 输出 u0(k)
                                         │
    生成多项式:                           │
    G1 = 7 (oct) = 111 (bin) = 1 + D + D² → u1 = d ⊕ S1 ⊕ S2
    G0 = 5 (oct) = 101 (bin) = 1 + D²     → u0 = d ⊕ S2

    尾比特处理: convenc() 自动追加 K-1=2 个零比特使编码器归零
```

### 4.2 状态转移图

```
    状态 (S1,S2)      输入=0              输入=1
    ─────────────   ───────────        ───────────
       00 (a)       → 00 / 00          → 10 / 11
       01 (b)       → 00 / 11          → 10 / 00
       10 (c)       → 01 / 10          → 11 / 01
       11 (d)       → 01 / 01          → 11 / 10
                    (下一状态/输出)
```

### 4.3 块交织/解交织操作

```
    交织 (写入→读出):
    ┌────────────────────────────────────────┐
    │  写入: 按行填充矩阵 (50行 × 80列)       │
    │                                        │
    │  ┌────┬────┬────┬────┬────┐           │
    │  │ b0 │ b1 │... │b79 │    │ ← 第0行   │
    │  ├────┼────┼────┼────┼────┤           │
    │  │b80 │b81 │... │b159│    │ ← 第1行   │
    │  ├────┼────┼────┼────┼────┤           │
    │  │ ...│    │    │    │    │           │
    │  └────┴────┴────┴────┴────┘           │
    │                                        │
    │  读出: 按列读出 (b0, b80, b160,...)    │
    └────────────────────────────────────────┘

    解交织 (写入→读出):
    ┌────────────────────────────────────────┐
    │  写入: 按列填充矩阵 (50行 × 80列)       │
    │  读出: 按行读出                        │
    └────────────────────────────────────────┘
```

## 五、系统关键参数汇总

| 参数 | 值 | 说明 |
|------|-----|------|
| Mfsk | 4 | 4FSK 调制 |
| 比特率 BR | 2 kbps | Mlog2 × 1k = 2×1k |
| 符号率 | 1 kSps | BR / Mlog2 |
| 调制指数 h | 0.5 | F_dev × Tsym = 0.5 |
| 频偏 F_dev | 500 Hz | h / Tsym |
| 带宽-时间积 BT | 0.5 | 高斯滤波器参数 |
| 射频采样率 fs | 32 MSps | DAC采样率 |
| 发射采样率 fs_tx | 64 kSps | sps=16 |
| 接收采样率 fs_rx | 16 kSps | 16×BW |
| 中频 Flo | 500 kHz | IF频率 |
| 载波频率 | 433.92 MHz | RF载波 |
| 卷积码 | (7,5) | K=3, R=1/2 |
| 交织器 | 50×80 | 4000 bit |
| 维特比回溯 | tblen=15 | 5×K |

## 六、BER统计方法对比

| 模式 | 比较对象 | 有效比特数 | 备注 |
|------|---------|-----------|------|
| 未编码 | tx_bits ↔ rx_bits | Nsym×2 = 4000 | 跳过帧头/帧尾 |
| 编码+硬判决 | info_bits ↔ decoded | 1998 | 维特比译码后比较 |
| 编码+MLSE | info_bits ↔ decoded | 1998 | ISI-aware Viterbi |
